    Noldor API v1.0.0

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    This is the Noldor API for sending data to and querying data from the Noldor platform.

    Base URLs:

    • https://api.noldor.com

    • https://api.nldrplayground.net

    Email: Noldor Web: Noldor

    Authentication

    • HTTP Authentication, scheme: bearer

    Authenticate

    Code samples

    import requests
headers = {
  'Content-Type': 'application/json',
  'Accept': 'application/json'
}

r = requests.post('https://api.noldor.com/v1/auth/client', headers = headers)

print(r.json())


    const inputBody = '{
  "id": "string",
  "secret": "string"
}';
const headers = {
  'Content-Type':'application/json',
  'Accept':'application/json'
};

fetch('https://api.noldor.com/v1/auth/client',
{
  method: 'POST',
  body: inputBody,
  headers: headers
})
.then(function(res) {
    return res.json();
}).then(function(body) {
    console.log(body);
});


    # You can also use wget

  -H 'Accept: application/json'

    POST /v1/auth/client

    Authenticate with service and retrieve an access token required for other API calls.

    Body parameter

    {
  "id": "string",
  "secret": "string"
}

    Parameters

    Name In Type Required Description
    body body object false none
    » id body string false none
    » secret body string false none

    Example responses

    200 Response

    {
  "access_token": "string",
  "expires": "string",
  "refresh_token": "string",
  "token_type": "string"
}

    Status Meaning Description Schema

    Body parameter

    {
  "reportName": "string",

  "reportFormat": "string",
  "reportFormatParameters": {}
}

    Parameters

    Name In Type Required Description
    body body object false none
    » reportName body string true none

    Responses

    Status Code 200
